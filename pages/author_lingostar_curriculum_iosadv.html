iOS 개발 고급

App prototyping

OS X 개발

  • 개발 과정명 : iOS 개발 고급
  • 교육시간 : 선택적 구성

1. 과정개요

  • iOS기반의 앱을 개발하는 보다 발전된 방법과 개발 방법론을 익힌다.
  • Touch와 Event처리에 대해 배움으로서 사용자 이벤트를 처리하는 화면 구성요소들에 대해 이해한다.
  • 앱 개발 시 점점 중요해지는 유닛 테스트에 대해 알아보고, 테스트 주도 개발 방법론을 실습해 본다.

2. 학습목차

01Touch와 Event

  • Touch & UIResponder [3시간]
    UITouch의 동작방식에 대해 익히고 UIView의 슈퍼클래스인 UIResponder의 메소드 오버라이드 방법을 알아본다.
  • Graphics [3시간]
    iOS에서의 Graphics 프레임워크에 대해 이해하고 기본적인 드로잉을 구현해 본다.
  • Custom Button 제작 [2시간]
    Custom 컨트롤 클래스를 만들어 봄으로서 UI의 완성도를 위한 작업방식을 배운다.
  • SignPad 만들기 [2시간]
    UITouch와 UIKit Graphics를 이용해서 간단한 사인패드 애플리케이션을 만들어본다. 개인별로 스트로크 컬러, 스트로크 두께, 그리고 알파값을 조절할 수 있는 기능을 추가해본다.

02Graphics의 핵심

  • UIControl & Gesture [3시간]
    Control 오브젝트의 추상 슈퍼클래스인 UIControl과 다른 방식의 이벤트 처리 방식인 UIGestureRecognizer를 이해한다.
  • Custom Button 제작 [2시간]
    Custom 컨트롤 클래스를 만들어 봄으로서 UI의 완성도를 위한 작업방식을 배운다.
  • Animation [2시간]
    다이나믹한 UI제작을 위해 iOS의 애니메이션 방법을 배운다.
    QuartsCore 레벨에서 뷰의콘텐트제공방식에 대해 이해한다.
  • 시계 버튼 만들기 [3시간]
    커스텀 컨트롤 제작과 애니메이션을 활용하여 시계처럼 동작하는 버튼을 만들어 본다.

03Test

  • 단위 테스트 방법론 [2시간]
    애자일 개발 방법론 중 하나인 단위 테스트 방법론에 대해 알아보고, 개발 과정에서 활용법에 대해 고민해본다.
  • Xcode에서의 TDD 개념 [3시간]
    Xcode에서 제공하는 OCUnit의 사용법을 배운다.
  • Instrument의 사용법 [3시간]
    Xcode에서 기본으로 제공하는 퍼포먼스 측정 도구인 Instrument의 사용법을 배운다.

04iOS의 기본 MDM

  • iOS의 MDM [3시간]
    애플이 제공하는 기본 MDM의 방식을 알아보고 실습해본다.
  • 등록과 배포 [3시간]
    Apple Developer Connection의 개발자 / 기기 / App ID의 관리 방법과 인증서 관리 / 프로비저닝 프로파일 관리 방법 실습
