App Name : 서울대 1팀
Creator : 장수연, 류원용
ADS
Purpose : 관심있는 문제 또는사건('이슈')의 최신 소식을 빠르게 확인하고 이슈의 전개 과정을 한눈에 파악할 수 있도록 함
Whos it's for : 모바일 기기로 뉴스를 주로 소비하는 통학/통근자 How they'll use it : 대중교통 안에서 비교적 짧은 시간동안 관심사를 확인하는 데 사용 Core Functionality : 관심있는 이슈의 최신 내용을 빠르게 확인하고, 개별 이슈가 지금까지 어떤 흐름으로 전개되어 왔는지를 볼 수 있음 Consolidated Definition Statement : 통학/통근자들이 짧은 시간동안 관심있는 이슈의 소식을 확인하고 새로 접하는 이슈의 전개 과정을 파악할 수 있도록 도와주는 어플리케이션
