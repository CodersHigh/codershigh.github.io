오토레이아웃 튜터 앱

Auto Layout Tutor의 베타 테스트에 참가해 주셔서 감사합니다.

혼자 취미처럼 만들다가 막상 오픈 베타를 하려니 부족한 부분이 너무 많이 보이네요. 그래도 일단 지금 단계라면 의견을 들을 수 있을 정도라 생각되어서 테스트를 부탁드립니다.

주변에 오토레이아웃에 관심 있는 분들에게도 테스트를 부탁해 주시면 고맙겠습니다.

스크립트 설치

애플리케이션이 사용하는 스크립트 폴더를 만들어야 합니다.

Home > Library > Application Scripts>com.codershigh.AutoLayoutTutor

폴더를 만들고 다운로드 받은 압축 파일을 풀어 GitScripts.scpt 와 XcodeScripts.scpt 를 넣어 주시면 됩니다.

자동으로 설치할 수 있으면 좋겠지만, Sandbox 규정상 Application Scripts 폴더에는 사용자만 쓰기 권한이 있다네요. 나중에는 Automator로 자동화 하도록 하겠습니다.

앱의 구동방식이나 콘텐츠에 대해 의견 있으면 얼마든지 이슈를 만들어 주세요.

누구나 자유롭게 이슈를 만들 수 있도록 열어 둔 이유는 제가 얼굴을 모르는 다수의 사용자들로부터 다양한 의견을 얻기 위해서입니다.

사소한 개선점이라도 보이면 메모장에 쓴다 생각하고 대충이라도 적어주시면 앱 개선에 도움이 됩니다.
