OS X 개발

  • 개발 과정명 : OS X 개발
  • 교육시간 : 선택적 구성

1. 과정개요

  • OS X 기반의 앱을 개발하고 배포하는데 있어서 iOS와의 차이점을 중심으로 알아본다.
  • 특히 iOS에는 없거나 개념이 다른 UI 요소들 - Window, Menubar, Dock 등과 플랫폼적인 특성으로 가능한 부분들을 익힌다.

2. 학습목차

01OS X UI

  • OS X의 Foundation [3시간]
    iOS에는 없고 OS X에만 있는 NSTree, NSXMLElement, NSArrayController와 같은 클래스들의 특성과 용도를 알아본다.
  • OS X의 UI 요소들 [3시간]
    NSTableView, NSOutlineView 등 OS X에서만 사용하는 UI들의 활용법을 알아본다.

02Background Process

  • Daemon과 Agent [3시간]
    간단한 동작을 하는 Daemon과 Agent를 만들어서 실행시키는 방법을 알아본다.
  • IPC [3시간]
    Daemon과 UI를 가진 App간의 데이터 통신방법을 실습해본다.

03AppleScript

  • Apple Script의 개요 [3시간]
    Apple Script의 개념과 동작 범위를 알아본다.
  • Apple Script 활용 [3시간]
    Apple Script를 이용해 할 수 있는 작업을 알아보고 우리의 앱이 Apple Script에 대응하도록 한다.

04배포

  • OS X 용 Installer 만들기 [3시간]
    OS X 용 인스톨러 프로그램을 이용해 Daemon을 설치하는 실습을 해본다.
  • Install 화면 구성과 Shell 사용 [3시간]
    인스톨러 제품을 만들기 위한 다양한 기능을 실습해본다.
